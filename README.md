## Setup

Установить зависимости проекта

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run start
npm run server
```

### Запуск unit тестов

```bash
npm run test:unit
```

## E2E

Использован [cypress](https://docs.cypress.io/guides/getting-started/writing-your-first-test.html#Add-a-test-file)

### Режим разработки (UI window)

```bash
npm run cy:open
```

### Запуск тестов

```bash
npm run cy:run
```
